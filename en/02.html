This unit was all about learning the beginnings of coding and conventional web design. We were taught to design mobile first, using responsive web design and media queries to make our projects work seamlessly on both mobile and desktop devices.

Our first introduction into design psychology was through the Gestalt laws, which we learned to employ in our designing in order to know when to obey the laws, and when to intentionally break them.

The final project for this theme was a simple website about computers, which we made in HTML and CSS based on a wireframe and content we were given. This was also our uptake activity exam.
